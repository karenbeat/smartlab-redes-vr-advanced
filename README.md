# SmartLab Redes VR Advanced - UNB

Este repositório reúne a documentação técnica do projeto **SmartLab Redes VR Advanced**, desenvolvido para a atividade de Metaverso Avançado da Residência em TIC 29.

O projeto propõe uma sala de aula inteligente, imersiva e colaborativa, inspirada no Lab Redes da Universidade de Brasília. A proposta apresenta um ambiente virtual voltado ao ensino de redes de computadores, com servidor didático, computadores dos alunos, estação do professor, televisões interativas, painéis de topologia e elementos de interação em Realidade Virtual.

## Situação do desenvolvimento

Durante a tentativa de uso do Unity Editor, o software apresentou falha de inicialização e encerramento inesperado durante a abertura. Acredita-se que o problema possa estar relacionado à incompatibilidade do sistema operacional ou a limitações do ambiente local de execução.

Por esse motivo, não foi possível avançar para a implementação prática da cena no editor. Assim, a entrega foi estruturada como um relatório técnico avançado, descrevendo o planejamento completo do ambiente, os recursos previstos, a organização da cena, o sistema de interação, o script C# proposto e a expansão multiplayer planejada.

## Objetivo do projeto

O objetivo do SmartLab Redes VR Advanced é representar um laboratório de redes em um ambiente de Metaverso educacional. A experiência foi planejada para permitir que estudantes explorem equipamentos, compreendam a função de servidores e visualizem conceitos de infraestrutura de rede de forma imersiva.

A versão avançada do projeto também considera a possibilidade de colaboração entre múltiplos usuários por meio de uma proposta de integração com Photon Fusion 2, permitindo que diferentes estudantes acessem o mesmo ambiente virtual e interajam com objetos didáticos.

## Recursos planejados

- Ambiente 3D inspirado no Lab Redes da UnB
- Servidor didático interativo
- Computadores futuristas para os alunos
- Estação do professor
- Televisões e painéis informativos
- UI em World Space para explicações dentro do ambiente
- Interação com objetos por raycast/controlador
- Script C# para ativação de painel informativo
- Proposta de expansão multiplayer com Photon Fusion 2
